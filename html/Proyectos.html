Proyecto La ponderosa

El proyecto LA PONDEROSA, se encuentra ubicado en la vereda Santa Barba corregimiento El Triunfo Este proyecto consta en la contrucción de un tramo de línea de media tensión con subestacion de 37.5 Kva, para el suministro de energía de 7 usuarios.

Normalización Red Baja Tensión

Este proyecto se encuentra ubicado en el corregimiento El Juncal del municipio Palermo. En este proyecto se ejecutó la georreferenciación, instalación de postería en baja tensión de 8 metros con cableado trenzado 2*35/77 e iluminación pública.

Instalaciones Internas
